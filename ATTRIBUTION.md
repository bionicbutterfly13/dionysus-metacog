## Usage Rule

ALWAYS treat the prototype as cited lineage and design evidence unless a
separate licensing review confirms code reuse is allowed.

If any prototype code is copied, translated, or substantially adapted into this
package later:

1. Record the exact source file and commit.
2. Preserve the original author attribution.
3. Add the required license notice before release.
4. Add a source note in `docs/source-inventory.md`.
5. Add or update tests that prove the adapted behavior.

Until those steps happen, references to the prototype in this repository are
documentation and design provenance only, not incorporated source code.
